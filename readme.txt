the User Agent Displayer plugin displays the Browser and the operating system, user used to leave a comment.

== Description ==

this plug-in displays the Browser and Platform of user who commented in your blog. it is capable of determining the version of the borwser.
== Installation ==

1. Upload `user-agent-displayer` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. thats it.
